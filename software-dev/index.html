Other Stuff

Helpful Shell Scripts - This repo just has a few helpful bash scripts such as a script that geo-locates you and updates your slack status based on location. There's a script that displays colorcoded timezone information in your terminal. A script that helps you test bandwidth between two machines. etc, etc ....

iloss-ical-gen - The purpose of the application is to provide an ICS calendar of projected weight loss. The user can recreate the calendar anytime by visiting the page to get a new url or just manipulating the url parameters to values of his/her choosing. This is available for use right here on my site.

Exchange Rate Alfred workflow - This workflow enables you to check exchange rates in Alfred (Mac automation tool) quickly by keying in: "exr" in the Alfred window.

IP Address Information Alfred workflow - Use this to see your local IP address information as well as your internet facing address via Alfred.

Jnetlib - My Cisco TACACs implementation from many moons ago.
